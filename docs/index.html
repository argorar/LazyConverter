Key features

A compact set of tools for conversion and light editing.

Flexible conversion

Export to MP4, MKV, and other common formats. Adjust resolution (original, 4K, 1080p, 720p, etc.).

Interpolation

Improve motion smoothness for cleaner playback.

Speed and trimming

Change speed between 0.5x and 2x with real-time preview. Set trim start and end using the current playback time.

Visual crop

On-screen overlay with handles to define the export area. Normalized coordinates for any resolution.

Auto analysis

Automatically detects resolution, duration, file size, and FPS when you load a video.

Smart preview

Native AVPlayer integration with time readout and crop overlay when enabled.

Dynamic speed

Adjust video speed dynamically over time using keyframes.

Dynamic crop

Animate your crop area to reframe shots or track subjects.

Dynamic zoom

Create smooth pan and zoom effects for a cinematic feel.

Watermark

Add customized text to your videos.

Fast workflow

Three steps from raw file to final export.

1

Drag a video in or select the file.

2

Adjust format, resolution, quality, and speed, trim, or crop options.

3

Convert and receive a notification when the process finishes.

Requirements

To run this app you need to have FFmpeg installed on your system.

Execute the next commands to install them:

brew tap homebrew-ffmpeg/ffmpeg
brew install homebrew-ffmpeg/ffmpeg/ffmpeg --with-libvidstab
